Class: AbstractNDArray

jsnum~ AbstractNDArray

new AbstractNDArray

A base class for n-dimensional arrays that provides a rich set of functionality building on a small set of functions required in derived classes. To create a new n-D array class, you can simply create a new class inheriting from this one that implements getElement, shape, and (for writable arrays) setElement.

at

Create a new n-D array object that provides a different view of the data in this array based on the indexes provided. Specifying an integer for the index for a given dimension will bind that index to that dimension in the new view, for example picking out a particular row or column of the matrix (thus lowering the number of dimensions by 1). Giving a string containing a number will create a new "dummy" dimension of the specified size. Giving an Array with two elements will select a subrange of the array along that dimension.

householderTransform

Compute the Householder transformation that will zero all but the first element of this vector. The transformation itself is a reflection across a hyperplane that passes through the origin; thus this reflection can be represented by an orthogonal matrix P which applies the transform or by a vector v that is normal to the hyperplane. This function returns a vector v, a matrix P, and a scalar beta such that v[0] = 1, P = I - beta v v^t, and P.dot(this) has zeros everywhere except for the first element. This transformation is usually used as a building block for other transforms and decompositions. Based on algorithm 5.1.1 in Golub GH, Van Loan CF. Matrix Computations. 3rd ed. The Johns Hopkins University Press; 1996.

Decomposes this matrix into three matrices, P, L and U, such that P is a row permutation matrix, L (lower) has no elements above the diagonal, U (upper) has no elements below the diagonal, and P L U = A.
